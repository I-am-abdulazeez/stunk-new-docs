Stunk is a modern state management library designed for simplicity and performance. Built on atomic state principles, it breaks your state into manageable "chunks" for fine-grained reactivity.

## Why Stunk?

- **Fine-grained reactivity** - Only affected components re-render
- **Framework agnostic** - Works with React, Vue, Svelte, and vanilla JS
- **Simple API** - Intuitive chunk-based state management
- **TypeScript first** - Full type safety out of the box
- **Async support** - Built-in handling for async operations

## Basic Example

```tsx
import { chunk } from "stunk";
import { useChunk } from "stunk/react";

// Create a chunk
const counterChunk = chunk(0);

// Use in React component
function Counter() {
  const [count, setCount] = useChunk(counterChunk);

  return <button onClick={() => setCount(count + 1)}>Count: {count}</button>;
}
```
